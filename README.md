Adoptapet

Un directorio digital interactivo para un refugio de animales, la aplicación muestra mascotas en adopción, permite filtrar por especie, buscar por nombre y resaltar los casos de adopción urgente.

-Ejecutar localmente

-paso 1. Instala las dependencias:

npm install

-paso 2. Inicia el servidor de desarrollo:

npm run dev

-paso 3. Abre la dirección que indica Vite en tu navegador.

-GitHub Copilot

Este proyecto fue asistido por GitHub Copilot para generar y estructurar el código de la aplicación.
